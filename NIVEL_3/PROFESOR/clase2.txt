import pandas as pd
import io

def generar_resumen_por_apellido(csv_content: str) -> pd.DataFrame:
    """
    Procesa el contenido CSV, agrupa los contactos por 'Apellido', 
    y cuenta la frecuencia de cada apellido.

    Args:
        csv_content (str): Contenido completo del archivo contactos.csv.

    Returns:
        pd.DataFrame: Un DataFrame con el conteo de contactos por apellido,
                      ordenado de forma descendente.
    """
    # 1. Leer el contenido del CSV directamente desde el string
    data_io = io.StringIO(csv_content)
    df = pd.read_csv(data_io)

    # 2. Eliminar duplicados (por si acaso, aunque no es el foco principal)
    df.drop_duplicates(inplace=True)

    # 3. Agrupar por 'Apellido' y contar la ocurrencia de cada uno
    resumen_df = df.groupby('Apellido').size().reset_index(name='Total_Contactos')

    # 4. Ordenar el resumen de forma descendente por el conteo
    resumen_df = resumen_df.sort_values(by='Total_Contactos', ascending=False).reset_index(drop=True)

    return resumen_df

Instalar y configurar un **entorno virtual** de Python en Windows es una práctica esencial para el desarrollo, ya que aísla las dependencias de tus proyectos. Esto se hace principalmente utilizando el módulo estándar de Python llamado `venv`.

Aquí tienes la guía paso a paso para crear y activar un entorno virtual en Windows utilizando el **Símbolo del Sistema (CMD)** o **PowerShell**.

-----

## 1\. Verificar la Instalación de Python

Asegúrate de que Python ya está instalado en tu sistema y accesible desde la línea de comandos:

1.  Abre el **Símbolo del Sistema (CMD)** o **PowerShell**.

2.  Escribe el siguiente comando y presiona Enter:

    ```bash
    python --version
    ```

    o

    ```bash
    py --version
    ```

Si ves un número de versión (ej. `Python 3.10.6`), puedes continuar. Si obtienes un error, necesitas instalar Python primero.

-----

## 2\. Crear el Entorno Virtual

Ve a la carpeta de tu proyecto (o la carpeta donde quieres que se almacene el entorno):

1.  Usa el comando `cd` (Change Directory) para navegar a la carpeta de tu proyecto. Por ejemplo:

    ```bash
    cd C:\Users\TuUsuario\Documentos\MiProyecto
    ```

2.  Crea el entorno virtual usando el módulo `venv`. El nombre `venv` (o `env`) es la convención más común, pero puedes usar el que quieras.

    ```bash
    python -m venv venv
    ```

      * **`python -m venv`**: Le dice a Python que ejecute el módulo `venv`.
      * **`venv` (segunda aparición)**: Es el nombre de la carpeta que contendrá el entorno virtual (los binarios de Python, `pip`, etc.).

    Este proceso creará una carpeta llamada `venv` dentro de tu proyecto.

-----

## 3\. Activar el Entorno Virtual

Una vez creado, debes "activar" el entorno para que tu sistema comience a usar los binarios de Python y las librerías instaladas dentro de esa carpeta, en lugar de las del sistema.

El comando de activación varía ligeramente dependiendo de si estás usando **Símbolo del Sistema (CMD)** o **PowerShell**.

### A. Para Símbolo del Sistema (CMD)

```bash
venv\Scripts\activate
```

### B. Para PowerShell

```bash
.\venv\Scripts\Activate.ps1
```

*(**Nota sobre PowerShell:** Si ves un error relacionado con la política de ejecución, es posible que debas ejecutar temporalmente `Set-ExecutionPolicy RemoteSigned -Scope Process` y luego reintentar la activación. Esto es por seguridad de Windows, pero es común en entornos de desarrollo.)*

### 🟢 Confirmación de Activación

Si la activación fue exitosa, verás el nombre de tu entorno virtual entre paréntesis al inicio de la línea de comandos, así:

```bash
(venv) C:\Users\TuUsuario\Documentos\MiProyecto>
```

-----

## 4\. Usar e Instalar Librerías

Mientras el entorno esté activo, cualquier librería que instales con `pip` se guardará solo dentro de este proyecto.

1.  Instala una librería de ejemplo:

    ```bash
    pip install requests
    ```

2.  Verifica la librería instalada (solo para este entorno):

    ```bash
    pip freeze
    ```

-----

## 5\. Desactivar el Entorno

Cuando termines de trabajar en el proyecto, simplemente usa el comando:

```bash
deactivate
```

El nombre `(venv)` desaparecerá de la línea de comandos, indicando que has vuelto a usar el Python instalado a nivel global en tu sistema.
